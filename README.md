# Java Learning

Учебный репозиторий, в котором я осваиваю основы Java
в рамках перехода из manual QA в автоматизацию тестирования.

## Что здесь есть

- **Main.java** — первые программы: вывод в консоль,
  переменные, арифметические операции.
- **FizzBuzz.java** — работа с циклами и условиями:
  классическая задача FizzBuzz, проверка чётности,
  подсчёт суммы чисел (паттерн «накопитель»).

## Изученные темы

- Синтаксис Java, переменные и типы данных (`int`, `String`)
- Арифметические операции, оператор остатка `%`
- Условия: `if / else if / else`, логические операторы (`&&`)
- Циклы `for`
- Работа с Git: коммиты, история изменений
